📊 Fully Automatic HR Payroll Management System (Excel)

A fully automated HR Payroll Management System built in Microsoft Excel to streamline employee attendance tracking, salary calculation, and payroll processing.

🔑 Features:
• Automated Attendance Sheet for tracking employee presence, absences, and working days.
• Dynamic Salary Sheet with automatic salary calculations based on attendance and predefined salary structures.
• Auto-generated Salary Slips for each employee.
• Centralized employee payroll records for easy management.
• Error-free calculations using Excel formulas and automation.
• User-friendly dashboard for efficient HR and payroll operations.

📈 Key Benefits:
• Reduces manual payroll work and calculation errors.
• Saves time in salary processing.
• Improves payroll accuracy and transparency.
• Simplifies employee record management.

🛠️ Tools Used:
• Microsoft Excel
• Advanced Excel Formulas
• Data Validation
• Conditional Formatting
• Pivot Tables
• Automated Salary Slip Generation
